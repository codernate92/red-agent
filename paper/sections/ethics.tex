\section{Ethics and Responsible Disclosure}
\label{sec:ethics}

This project is framed as authorized security research. The framework is built
to test model behavior through documented APIs and local/open-weight endpoints,
not to access unrelated systems or exploit third-party infrastructure. The
repository includes provider-specific and generic disclosure templates in
\path{engine/templates/} and a \texttt{FindingRouter} that turns confirmed
findings into review or disclosure actions.

The paper also reflects a deliberate reporting choice. We include concrete trace
excerpts because they are necessary to substantiate the observed jailbreak
behavior, but we avoid turning the manuscript into a prompt cookbook. The most
actionable operational details remain in the code and artifact bundle, which is
appropriate for a research framework whose intended users are evaluators rather
than casual attackers.

Operational reachability raises an ethical point as well. A provider access
failure is not a model vulnerability, and the paper does not present it as one.
Instead, preflight status is reported as an execution condition that determines
what can be tested and how confidently claims can be generalized.
