In this run, all configured aliases are reachable in preflight. This improves
comparability against prior broader registries where execution blockers
(credential failures, quota limits, or stale model IDs) obscured model-level
interpretation.

Reachability should still be treated as a first-class result. The absence of
blockers in one run does not eliminate operational risk in later runs; it only
means this specific artifact set reflects a fully executable configured cohort.
